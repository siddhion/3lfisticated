Hello my name is Adam and my designer name is 3lfist. While I used to primarily focus on graphic design and illustration, my main focuses now are UX design and web development. I am currently designing and developing my own gamified photo sharing app called 3lf using web technologies like AngularJS, Firebase, Git, Foundation 5, CSS3, HTML5 and SASS. It's currently not deployed but the code can be viewed at Github here:https://github.com/3lfist/3lf

My design process starts out with me hand drawing mockups of the app. This includes user interactions, page flow and many notes on the behaviour of the app.

My approach to application UX is holistic. I take into consideration almost every aspect of the product :

  • Branding
  • Email Design
  • Page Composition
  • Core Functionality
  • Ease of use
  • Responsiveness (RWD)
  • Cross Browser Compatibility
  • Security
  • Page Load Time Optimization

I made the first version of this site in 2012 using AngularJS 1.0.6, Susy, Compass, SASS, HTML5 and CSS3. Few things changed but it's basically the same code and design.
